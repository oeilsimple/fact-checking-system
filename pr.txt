Build a **premium, AI-powered fact-checking platform interface** using Next.js 14+ and Tailwind CSS. This is the frontend for "TruthBot," a sophisticated multi-agent AI system that fact-checks claims in real-time using web research and AI reasoning. The design should feel **modern, trustworthy, and visually striking**‚Äîbalancing scientific credibility with sleek tech aesthetics.

üõ†Ô∏è TECHNICAL IMPLEMENTATION REQUIREMENTS

**Stack**:
- **Framework**: Next.js 14+ (App Router recommended)
- **Styling**: Tailwind CSS 3.4+
- **UI Components**: Headless UI, Radix UI, or shadcn/ui (optional but recommended)
- **Markdown Rendering**: react-markdown + remark/rehype
- **HTTP Client**: fetch API or axios
- **Loading States**: React hooks or Suspense
- **Toast Notifications**: React Hot Toast or Sonner
- **Icons**: Heroicons, Lucide, or Tabler Icons
- **Animations**: Framer Motion (optional, for advanced animations)

**Environment Setup**:
- `.env.local` file with API base URL:
  ```
  NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
  # or your production API endpoint
  ```

**API Integration**:
```javascript
// Example fetch to /fact-check endpoint
const response = await fetch(
  `${process.env.NEXT_PUBLIC_API_BASE_URL}/fact-check`,
  {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ claim: userInput })
  }
);
const data = await response.json();
// data.verdict is markdown string
// Parse and render with react-markdown
```

**State Management**:
- Use React hooks (useState, useEffect) for simple state
- Or Zustand/Jotai if complexity grows

**Performance**:
- Optimize images (next/image)
- Lazy load markdown renderer
- Debounce input (optional)
